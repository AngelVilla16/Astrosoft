🎯 Objetivo Principal

ASTRA busca ofrecer una solución automatizada e intuitiva para clínicas pequeñas o consultorios, facilitando el registro, la consulta y la gestión de:

  • Información de pacientes.
  • Citas médicas.
  • Expedientes clínicos.
  • Usuarios del sistema (personal médico, recepcionistas, administradores).

🧱 Componentes del Sistema

Interfaz Gráfica (Windows Forms)

Una interfaz de usuario amigable y funcional, diseñada para el usuario final.

Base de Datos Local (SQL)

Sistema de almacenamiento persistente, embebido dentro de la aplicación.

Módulo de Gestión

  • Registro e inicio de sesión de usuarios (con contraseñas cifradas en SHA-256).
  • Operaciones CRUD (Crear, Leer, Actualizar, Eliminar) para pacientes.
  • Registro y control de citas.
  • Gestión de expedientes médicos (prescripciones, historial, etc.).

👤 Perfiles de Usuario

Recepcionista

Encargado de agendar, consultar y cancelar citas.

Médico

Visualiza historial, receta medicamentos. También realiza las funciones de recepcionista y administrador.

Administrador

Gestiona usuarios y configuración general.

Paciente (futuro)

Podría acceder a su historial desde una app o portal.

🔐 Seguridad

  • Login Protegido: Las contraseñas de los usuarios se cifran utilizando el algoritmo SHA-256.
  • Validación de Credenciales: Verificación rigurosa de las credenciales al iniciar sesión.
