🎯 Objetivo Principal

NOVA está diseñada para ser una herramienta simple y eficaz para la gestión de inscripciones a talleres en instituciones educativas, centros comunitarios o cualquier organización que necesite una forma rápida de registrar y organizar a sus estudiantes. Su enfoque está en la facilidad de uso y en la funcionalidad principal de administrar la base de datos de alumnos y talleres de forma local.

🧱 Componentes del Sistema

Interfaz Gráfica (Windows Forms)

Una interfaz amigable para el registro y gestión de alumnos.

Base de Datos Local (LocalDB)

Un sistema de almacenamiento de datos ligero y embebido.

Módulo de Gestión de Alumnos

  • Registro de nuevos alumnos con sus datos.
  • Filtros por tipo de taller para una búsqueda eficiente.
  • Opción para remover alumnos del registro.

🔐 Seguridad

  • Integridad de Datos Local: La base de datos LocalDB asegura que la información de los alumnos se mantenga segura y accesible solo localmente en la computadora donde se instala la aplicación.

🧩 Tecnologías
